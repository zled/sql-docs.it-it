# <a name="getcharacterstream-method--sqlservernclob"></a>Metodo getCharacterStream () (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera le **NCLOB** i dati come una **lettore** oggetto o come un flusso di caratteri.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.io.Reader getCharacterStream()  
```  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="return-value"></a>Valore restituito  
 Oggetto Reader contenente i dati **NCLOB**.  
  
## <a name="remarks"></a>Remarks  
 Questo metodo getCharacterStream viene specificato dal metodo nell'interfaccia java.sql.NClob getCharacterStream.

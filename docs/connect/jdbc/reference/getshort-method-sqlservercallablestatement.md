# <a name="getshort-method-sqlservercallablestatement"></a>Metodo getShort (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **breve** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getShort (int)](../../../connect/jdbc/reference/getshort-method-int.md)|Recupera il valore del parametro designato come un **breve** in base all'indice del parametro di linguaggio di programmazione Java.|  
|[getShort (lang)](../../../connect/jdbc/reference/getshort-method-java-lang-string.md)|Recupera il valore del parametro designato come un **breve** nel linguaggio di programmazione Java specificato il nome del parametro.|

    -   [Attività Hive Hadoop](../integration-services/control-flow/hadoop-hive-task.md)

-   Flusso di dati: origine e destinazione dei dati  
  
    -   [Origine file HDFS](../integration-services/data-flow/hdfs-file-source.md)

# <a name="sql-server-cursor-manager-total-object"></a>Oggetto Cursor Manager Total di SQL Server
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  Tramite l'oggetto **SQLServer:Cursor Manager Total** sono disponibili contatori per il monitoraggio dei cursori.  
  
 Nella tabella seguente sono descritti i contatori dell'oggetto **Cursor Manager Total** di SQL Server.  
  
|Contatori di Cursor Manager Total|Description|  
|-----------------------------------|-----------------|  
|**Conteggio popolamento asincrono**|Numero di cursori popolati in modo asincrono.|  
|**Frequenza conversione cursori**|Numero di conversioni di cursore al secondo.|  
|**Scaricamenti cursore**|Numero totale di riformulazioni di istruzioni di run-time eseguite dai cursori.|

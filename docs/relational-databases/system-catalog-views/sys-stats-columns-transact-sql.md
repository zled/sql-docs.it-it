# <a name="sysstatscolumns-transact-sql"></a>sys.stats_columns (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-all-md](../../includes/tsql-appliesto-ss2008-all-md.md)]

  Contiene una riga per ogni colonna che fa parte di **Sys. Stats** statistiche.  
  
|Nome colonna|Tipo di dati|Description|  
|-----------------|---------------|-----------------|  
|**object_id**|**int**|ID dell'oggetto a cui la colonna appartiene.|  
|**stats_id**|**int**|ID delle statistiche a cui la colonna appartiene.<br /><br />Se le statistiche corrispondono a un indice, il *stats_id* valore Ã¨ identico il *index_id* valore il [Sys. Indexes](../../relational-databases/system-catalog-views/sys-indexes-transact-sql.md) vista del catalogo.|  
|**stats_column_id**|**int**|Ordinale in base 1 nel set di colonne di statistiche.|  
|**column_id**|**int**|ID della colonna da **Sys. Columns**.|

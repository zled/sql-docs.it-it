# <a name="mssqlserver7913"></a>MSSQLSERVER_7913
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>Dettagli  
  
|||  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|7913|  
|Origine evento|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nome simbolico|DBCC2_REPAIR_EXTENT_DEALLOCATED|  
|Testo del messaggio|Correzione: l'extent P_ID è stato deallocato dall'oggetto con ID O_ID, indice con ID I_ID, partizione con ID PN_ID, unità allocazione con ID A_ID (tipo TYPE).|  
  
## <a name="explanation"></a>Spiegazione  
Messaggio informativo inviato da REPAIR che indica che un extent è stato deallocato dall'oggetto specificato.  
  
## <a name="user-action"></a>Azione dell'utente  
None

# <a name="protocoldisplayname-property-servernetworkprotocol-class"></a>Proprietà ProtocolDisplayName (classe ServerNetworkProtocol)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Ottiene il nome visualizzato del server del protocollo di rete.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object.ProtocolDisplayName [= value]  
```  
  
## <a name="parts"></a>Parti  
 *oggetto*  
 A [classe ServerNetworkProtocol](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md) che rappresenta il protocollo di rete utilizzato dall'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore string che specifica il nome visualizzato del protocollo di rete del server.

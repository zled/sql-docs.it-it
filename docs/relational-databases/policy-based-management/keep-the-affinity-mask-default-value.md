# <a name="keep-the-affinity-mask-default-value"></a>Valore predefinito per la maschera di affinità
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Questa regola consente di controllare se l'opzione di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] per la maschera di affinità è impostata su 0, ovvero il valore predefinito. L'opzione relativa alla maschera di affinità determina in modo dinamico l'affinità della CPU.

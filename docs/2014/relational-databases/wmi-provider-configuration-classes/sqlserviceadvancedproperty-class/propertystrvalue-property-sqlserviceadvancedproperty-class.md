# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>Proprietà PropertyStrValue (classe SqlServiceAdvancedProperty)
  Ottiene il valore string della proprietà avanzata.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto [classe SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) oggetto che rappresenta una proprietà avanzata.  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore string che specifica il valore della proprietà avanzata.

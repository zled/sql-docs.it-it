# <a name="remove-columns-from-query-results-visual-database-tools"></a>Rimuovere colonne dai risultati di una query (Visual Database Tools)
  Se si utilizza una colonna nella query di selezione ma si preferisce escluderla dal set di risultati che verranno visualizzati (vale a dire, escluderla dall'elenco di campi selezionati nella query), sarà possibile rimuoverla dall'output. Dopo avere rimosso la colonna dall'output della query, sarà comunque possibile utilizzarla nelle condizioni di ricerca o come campo di ordinamento.  
  
> [!NOTE]  
>  Per rimuovere completamente una colonna dalla query, vedere [Rimozione di colonne da query &#40;Visual Database Tools&#41;](visual-database-tools.md).  
  
### <a name="to-remove-a-column-from-the-query-output"></a>Per rimuovere una colonna dall'output della query  
  
-   Nel **riquadro Criteri**deselezionare nella colonna **Output** la casella di controllo relativa alla colonna di dati da rimuovere. Per ripristinare la colonna nell'output della query, sarà possibile selezionare nuovamente la colonna **Output** .  
  
     oppure  
  
-   Rimuovere la colonna dall'elenco di output nel [riquadro SQL](sql-pane-visual-database-tools.md).

# <a name="programming-with-ado-md"></a>Programmazione con ADO MD
Per utilizzare ADO MD con lo strumento di sviluppo, è necessario stabilire un riferimento alla libreria dei tipi ADO MD. La descrizione della libreria ADO MD è una libreria di Microsoft ActiveX Data Objects (multidimensionali). Il nome di file di libreria ADO MD è msadomd e l'ID di programma (ProgID) è "ADOMD". Per ulteriori informazioni su come stabilire i riferimenti alle librerie, vedere la documentazione dello strumento di sviluppo.

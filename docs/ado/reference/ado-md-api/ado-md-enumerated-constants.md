# <a name="ado-md-enumerated-constants"></a>Costanti enumerate ADO MD
Per agevolare il debug, le costanti enumerate ADO MD elenca un valore per ciascuna costante. Tuttavia, questo valore è puramente consulenza e può variare da una versione di ADO MD a un altro. Il codice deve dipendere esclusivamente il nome, non il valore effettivo di costanti enumerate.  
  
 Le enumerazioni seguenti vengono definite.  
  
-   [SchemaObjectTypeEnum](../../../ado/reference/ado-md-api/schemaobjecttypeenum.md)  
  
-   [MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md)

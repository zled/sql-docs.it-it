# <a name="levelname-property-ado-md"></a>Proprietà LevelName (ADO MD)
Indica il nome del livello di un [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Valori restituiti  
 Restituisce un **stringa** ed è di sola lettura.  
  
## <a name="remarks"></a>Note  
 Usare la **LevelName** proprietà per recuperare il nome del livello a cui appartiene un membro. Corrisponde al [nome](../../../ado/reference/ado-md-api/name-property-ado-md.md) proprietà di un [livello](../../../ado/reference/ado-md-api/level-object-ado-md.md) oggetto.

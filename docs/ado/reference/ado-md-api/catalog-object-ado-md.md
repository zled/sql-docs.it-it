# <a name="catalog-object-ado-md"></a>Oggetto Catalog (ADO MD)
Contiene informazioni sullo schema multidimensionale (vale a dire, i cubi e sottostante dimensioni, gerarchie, livelli e membri) specifiche di un provider di dati multidimensionali (dati Multidimensionali).  
  
## <a name="remarks"></a>Note  
 Con le raccolte e le proprietà di un **catalogo** dell'oggetto, è possibile eseguire le operazioni seguenti:  
  
-   Aprire il catalogo impostando il [ActiveConnection](../../../ado/reference/ado-md-api/activeconnection-property-ado-md.md) proprietà di un oggetto ADO standard [connessione](../../../ado/reference/ado-api/connection-object-ado.md) oggetto o a una stringa di connessione valida.  
  
-   Identificare le **Catalog** con il [nome](../../../ado/reference/ado-md-api/name-property-ado-md.md) proprietà.  
  
-   Eseguire l'iterazione attraverso i cubi in un catalogo usando il [CubeDefs](../../../ado/reference/ado-md-api/cubedefs-collection-ado-md.md) raccolta.

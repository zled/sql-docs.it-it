# <a name="type-property-table-adox"></a>Proprietà Type (Table) (ADOX)
Indica il tipo di una tabella.  
  
## <a name="return-values"></a>Valori restituiti  
 Restituisce un **stringa** valore che specifica il tipo di tabella, ad esempio, "TABLE", "Tabella di sistema" o "Temporanei globali".  
  
## <a name="remarks"></a>Note  
 Questa proprietà è di sola lettura.

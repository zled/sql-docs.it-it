# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Supporto per Visual Studio Analyzer è stata rimossa a partire da Windows 8 (Visual Studio Analyzer è stato incluso solo nelle versioni precedenti di Visual Studio). Per una meccanismo di risoluzione dei problemi in alternativa, usare l'analisi BID.  
  
 Microsoft® Visual Studio™ Analyzer fornisce una panoramica generale dell'applicazione ODBC. Lo sviluppatore può valutare, analizzare e la struttura, prestazioni e le interazioni dal punto di vista dell'applicazione, invece che dal punto di vista del componente o codice di debug. Le informazioni raccolte da Visual Studio Analyzer sono sotto forma di eventi che rappresentano un tipo di interazione tra i due componenti dell'applicazione. Gli eventi possono essere composte di chiamate di funzione e restituisce da funzioni, ad esempio le connessioni, le query di database e transazioni.
